# RightEstate - Real Estate Search App

A modern React application for browsing and searching properties with advanced filters, drag-and-drop favourites, and detailed property views.

## Features
- Property search with price, bedroom, postcode, and type filters
- Drag properties to favourites panel
- Detailed property page with image gallery, booking form, map, and similar listings
- Fully responsive dark theme
- Unit tested components

Built with React, React Router, react-dnd, and react-tabs.
